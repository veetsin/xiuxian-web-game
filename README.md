# 青石问道录 · 轮回修仙

> 一镇，一山，一口气。死了再来。

一个**单地图、轮回、随机出生、隐藏道途**的修仙文字网页游戏。你不是开局选职业，而是在同一片凡界里一次次轮回——出生、行动、奇遇、战斗、死亡、记忆——逐渐悟出属于自己的修行路线，用前世的知识改这一世的命。

纯原生 HTML / CSS / JS，**零依赖、零构建**。双击 `index.html` 即玩。

---

## 如何游玩

打开后点「入世」，每一世从随机出生开始（猎户之子、病弱孤儿、药铺学徒、屠户学徒、武馆杂役、庙祝养子……）。

- **每个月选一个主要行动**：地图上不同地点（青石镇、黑山、山神庙、废弃矿洞、武馆、药铺、家中……）会根据你的状态、时节、记忆、隐藏倾向给出不同选项。
- **行为会暗中改变状态**：受伤后获胜、用剑杀敌、频繁服药、雷雨夜行动、用前世记忆改命……这些都在悄悄把你推向某条「道」。
- **道不会直接告诉你**。先是异象（「你受伤后，腰间剑柄微微发热」），多次触发、关键战斗或顿悟后才逐渐显名。顿悟时你可以**纳之**（接纳，显化道途、解锁神通）或**抑之**（压制，也是一种修行）。
- **战斗能让你感到自己在变强**：构筑触发会写进战斗日志（血意叠层、伤口倒流、追加斩击……），还有战果评价（险胜 / 压制 / 碾压 / 威压降服）。
- **前期有几个明显打不过的敌人**（黑山狼王、矿洞尸王、武馆大师兄、山神庙邪影）。死了不要紧——轮回会保留部分记忆、情报、道痕和称号残响。下一世带着前世的梦境与弱点情报，回去把它们打掉。

右侧面板是你的**悟道录 / 前世记忆 / 传闻 / 称号 / 行囊 / 人物**——它不是攻略菜单，而是你自己探索这个世界的笔记，随游玩逐渐补全。进度自动存档（浏览器 localStorage），随时「保存 / 读取 / 重开」。

一局 15 分钟以上。目标，是让你愿意说一句：「再轮回一世试试。」

## 如何运行

### 在线游玩
部署后直接访问公网地址即可（见下方「部署」）。

### 本地运行（任选其一）

**方式 A：直接打开**
双击项目根目录的 `index.html`，浏览器以 `file://` 打开即可游玩。

> 个别浏览器在 `file://` 下会禁用 localStorage（导致不能存档）。若遇到，用方式 B。

**方式 B：本地静态服务**
```bash
# 任选一个，在项目根目录执行
python3 -m http.server 8080
# 或
npx serve .
```
然后浏览器打开 `http://localhost:8080`。

无需 `npm install`，无需构建步骤——本项目没有任何运行时依赖。

## 部署（GitHub Pages）

纯静态站点，GitHub Pages 零配置即可托管：

1. 把本仓库推到 GitHub（公有仓库）。
2. 仓库 **Settings → Pages → Build and deployment → Source** 选 **Deploy from a branch**，分支选 `main`、目录选 `/ (root)`，保存。
3. 等一两分钟，公网地址形如 `https://<用户名>.github.io/xiuxian-web-game/`，把它发给朋友即可游玩。
4. 之后每次 `git push` 到 `main` 都会自动重新部署。

> 进度保存在**各人自己浏览器的 localStorage**里——纯前端、不联网、不上传，朋友的存档只在自己设备上。
>
> 也可换 Vercel / Netlify / Cloudflare Pages：同样零构建，导入仓库即部署。

## 核心系统

整个游戏围绕一条链设计：**行为改变状态 → 状态改变概率 → 概率生成事件 → 事件留下后果 → 后果改变行动 → 行动推动隐藏道 → 隐藏道改变战斗与选择 → 死亡留下记忆 → 记忆改变下一世策略。**

| 系统 | 说明 |
|---|---|
| **世界状态** | 六个世界变量（狼患 / 恐慌 / 阴气 / 矿洞不稳 / 仙门关注 / 药价）每月自走漂移；阈值触发世界事件（狼祸下山、矿洞塌方、庙中异动）。你长期闭关，世界也会变。 |
| **出生系统** | 6 种随机出生，决定初始属性、地点、人际、资源、早期困境、命签、潜在路线。轮回记忆与道痕会影响来世出生权重。 |
| **隐藏道途** | 5 条潜在道（血剑 / 丹药 / 雷法 / 炼体 / 因果），开局完全不告知。行为暗中累积倾向 → 异象 → 悟道录 → 顿悟显名 → 神通。每条道打法不同（以伤换爆发 / 续航资源 / 先手克邪 / 站撸回复 / 保命与情报针对）。 |
| **战斗** | 回合制；攻击 / 运气 / 守势 / 神通 / 逃走。异象钩子在 6 个时机触发构筑链；流血、毒、栈、威压降服、战果评价一应俱全。 |
| **轮回** | 死亡 → 走马灯（此生年表 + 可携带之物）→ 蒸馏（部分记忆、已确认悟道、道痕种子、称号残响、前世痕迹）→ 重新随机出生，同一张地图，世界重置但留有印记。 |
| **动态 NPC** | 老猎户、药铺老板、武馆大师兄、庙祝、外门巡使、劫道散修各有月度行为弧线：变强、失踪、抢走机缘、散布传闻、对你态度变化。 |
| **装逼反馈** | 你强不强，世界会承认：敌人夹尾后退 / 跪地求饶、镇民今昔对比的议论、称号与传闻、前世名号的残响共鸣。 |

## 技术说明

- 纯原生 HTML/CSS/JS，`<script>` 顺序加载，共享唯一全局 `window.G`；无 ES Module、无 CDN、无图片素材（视觉全用 CSS 渐变实现深墨 / 宣纸 / 朱砂 / 鎏金质感）。
- 数据驱动：出生 / 道 / 行动 / 事件 / 敌人 / NPC / 称号全部以 `G.define(...)` 注册，引擎用统一的条件 DSL 与效果 DSL 求值。
